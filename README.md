# PolicDog

由于个人喜欢玩红警, 所以取名自里面的警犬单位, 希望性能探测脚本可以像里面的警犬一样探测游戏地形与敌人方位, 探测 web 网页中性能有待优化的地方.

## web 性能指标

白屏时间

首屏时间

用户可操作时间

资源总下载时间

DNS 解析时间

TCP 连接时间

HTTP 请求时间

HTTP 响应时间

CSS 渲染时间

Javascript 渲染时间

## 为什么要关注这些指标
如果我们能够将网页加载的时间准确地获取, 那么我们就可以清晰地分析出网页加载性能的瓶颈, 从而解决瓶颈, 使网页秒开, 提升用户体验.

## 用法
可以得到关于网页加载的信息, 并上报到服务器上.
